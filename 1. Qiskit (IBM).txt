Tools of the Trade

Last week, we discussed several tools to program and use quantum computers, each having it's own benefits. Some allows us to use hashtag#QuantumAnnealers, some for hashtag#PhotonicsQuantumComputer, while others are specifically for hashtag#GateBasedQuantumComputers.

- D-Wave hashtag#Ocean: A set of open-source Python tools accessible for constructing, modifying and running quantum circuits using simulators or D-Wave's Quantum Annealers.
- IBM Quantum hashtag#Qiskit: Qiskit is software for real work on utility-scale quantum systems. Qiskit is built for powerful executions on IBM quantum systems.
- Xanadu hashtag#PennyLane: A Python library for quantum machine learning, automatic differentiation, and optimization of hybrid quantum-classical computations.
- Amazon hashtag#Bracket: A hashtag#CloudQuantumComputingService that helps researchers and developers use quantum computers and simulators to build quantum algorithms on AWS.
- Microsoft Azure hashtag#Quantum: Azure Quantum is a Microsoft Azure service that you can use to run quantum computing programs problems in the cloud.
- TensorFlow hashtag#Quantum: TensorFlow Quantum (TFQ) is a quantum machine learning library for rapid prototyping of hybrid quantum-classical ML models. It integrates quantum computing algorithms and logic designed in hashtag#Cirq, and provides quantum computing primitives compatible with existing hashtag#TensorFlow APIs.
